# coff Carpenters Office
quick calculation, estimation of costs and invoice
runs in a terminal


- file based storage for  
    + address
    + order
    + expense
- output: pdf and txt
- search function
- quick navigation by shortcut: type a to list adresses
- document types:
    + estimation of costs
    + order confirmation
    + delivery note
    + invoices


installation linux (debian based)
---
$ sudo apt install python3 python3-pip vim fzf
